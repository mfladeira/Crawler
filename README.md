# Crawler
Programa que dada uma URL, busca e faz o download de links e imagens disponíveis da URL principal e de seus sublinks.

## Requisitos

- [Node.js](https://nodejs.org/en/download/)

### Dependências

- [crawler](https://www.npmjs.com/package/crawler)

### Instalação
   
   `npm install`
   
### Execução

   `node crawler.js`
